Schemes

The color wheel is a visual representation of colors arranged according to their chromatic relationships.

There are 6 main color schemes.

  • Monochromatic: Combines different shades and tints of a single color.
  • Analogous: Pairs one color with two colors next to it on the color wheel.
  • Complementary: Uses two colors opposite each other on the color wheel.
  • Triadic: Combines three colors evenly spaced on the color wheel.
  • Split-complementary: Uses a base color and two adjacent to its complementary color.
  • Neutral: Utilizes neutral colors like black, white, and gray.
Hex

Hexadecimal color values, or hex codes represent colors in the RGB color model. It refers to the base-16 numbering system, which uses digits 0 to 9 and letters A to F. Below is an expanple of the layout.

element { color: #RRGGBB; }

Here is the hex code for a redish orange color

body { color:#FF5733; }

The precision and compatibility of Hex Codes allow for over 16million colors.

RGB

RGB stands for Red, Green, and Blue — the primary colors of light. These three colors are combined in different intensities to create a wide range of colors.

element { color: rgb(red, green, blue); }

These three colors are combined in different intensities to create a wide range of colors.

p { color: rgb(255, 0, 0); }

The above code would change the text color of the paragraph to red because the red value is set to 255, while green and blue are set to 0.

HSL

HSL stands for Hue, Saturation, and Lightness — three key components that define a color.

element { color: hsl(hue, saturation, lightness); }

The HSL color model represents colors in a way that is more in line with how humans perceive color

p { color: hsl(120, 100%, 50%); }
Gradients

CSS gradient functions let you display smooth transitions between two or more colors within an element.

Linear Gradients - The color transition goes down, up, left, right, or diagonally, Radial Gradients - The color transition goes out from a central point, Conic Gradients - The color transition is rotated around a center point

body { background:linear-gradient(to right, red, blue); }
